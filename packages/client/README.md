# What is it?

React frontend for adding and listing events. It consists of:

- [React](https://reactjs.org/)
- [tRPC](https://trpc.io)
- [TypeScript](https://typescriptlang.org)
- [Tailwind CSS](https://tailwindcss.com)
- [Vite](https://vitejs.dev/)

## Getting Started
Prerequisite:

- pnpm
- node 16.17.0

## Running locally

1. `pnpm install`
1. Run dev server - `pnpm dev`


## Running tests

```
pnpm jest
```
